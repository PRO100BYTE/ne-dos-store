# NE-DOS Store - каталог команд для NE-DOS


### С чего начать?
1. Выполните клонирование репозитория: \
   `git clone https://github.com/PRO100BYTE/ne-dos`
2. Установите зависимости: \
   `npm install`
3. Запустите сервер разработки: \
   `npm start`
